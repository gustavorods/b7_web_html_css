Tag "<ul>" serve para cria uma lista desordenada, e dentro dela vai a TAG "<li>" para os itens

3 formas de inserção do CSS no HTML

- CSS inline (<h1 style="color:red">Título da Página </h1)
OBS: também serve para mudar a fonte da letra a largura. 

- CSS interno (h1 { color:blue; }
OBS: o CSS fica na head cabeçalho. 

- CSS externo
No externo é necessário criar outro arquivo .CSS e e adicionar ele ao código, usando "<link rel="stylesheet" href="nome do outro arquivo" />".
Para aplicar cores e fontes diferentes em varias tags iguais é necessário criar id diferentes para cada um
ou criando uma "class", onde não é necessário colocar palavras diferentes para cada id.
Para colocar todos os items (id e class) com as mesmas cores e fontes basta colocar uma virgula.
exemplo: #titulo, .texto {
    color:cyan;
    font-size: 35px;
}
Para modificar varios itens que estão dentro de outro item ao msm tempo, não é necessário colocar a virgula. 
exemplo: ol li {
    color:pink;
}

todos os itens dentro de "ol" ficarão rosa.
OBS: só ficarão rosa os "li" que estão dentro do "ol", ou sela, se tiver mais de uma lista com (uma com ol e outra com ul) somente os da lista "ol" ficará rosa
