módulo 3 (aula10)
Select

Select: aquela opção de clicar e mostrar as opções disponiveis.
OBS: o select tem que ter um name.

Dentro do "select" vai colocar uma tag "option" e lá coloca as categorias.
Exemplo: mais vendidos, mais recentes, etc...

O option tem 2 versões de valores diferentes que pode ser usado, uma coisa é a que o usario ta vendo, outra coisa é o que vai ser enviado no formulário.
Exemplo: se colocar uma atributo "value" no "option" é possivel definir um valor específico para o item.
  

  <select name="order">
                <option valeu="bestseller">mais vendidos</option>
                <option value="last">mais recentes</option>
  </select>  

Apesar do usuário ter selecionada a opção "Mais vendidos", não necessariamente ele enviou aquele item em específico, aquele valor, o texto "mais vendidos", ele enviou o valor que eu botei como o valor padrão dessa opção, ou seja, quando eu não específico o "value" ele vai pegar o mesmo item que está sendo exibido, é o que vai ser enviado, e quando é colocado o "value" ele vai enviar esse valor, apesar de aparecer mais vendidos ou recentes. 


É possível pré selecionar algum item no HTML, então, quando eu não Preleciono nada, ele vai selecionar sempre o Primeiro.
Exemplo: 

 <select name="order">
                <option valeu="bestseller">mais vendidos</option>
                <option value="last">mais recentes</option>
  </select>  

Se eu não pré selecionar nada, o primeiro que vai aparecer é o "mais vendidos, porque ele está em primeiro no código.
E pra eu colocar o "mais recentes" em primeiro eu vou em "options" e adiciono um atributo "selected". 

<option value="bestseller">mais vendidos</option>
<option value="last" selected>mais recentes</option>

DICA: pra eu não pré selecionar nenhum item, ou seja, deixar em branco, é necessário colocar um "option" vazio em primeiro.
Exemplo:
 <select name="order">
                <option></option>
                <option valeu="bestseller">mais vendidos</option>
                <option value="last">mais recentes</option>
</select>

É possível transforma o select me múltipla escolha. 
Para fazer isso é necessário adicionar o atributo "Size" e "multiple" em select.
Size: o que vai determinar quantas opções eu posso marcar.
multiple: o que determinar que meu select seja de múltipla escolha

<select name="order" size="2" multiple>

Ai depois é só selecionar um item e segurar Ctrl (CONTROL) e clicar em outro.

-tudo que esta dentro do elemento, vai receber tal propriedade como herança.
Exemplo: se eu tenho uma div com a class="container", e colocar uma cor vermelha nela, tudo que estar dentro dela vai HERDAR a cor vermelha.

-Porém, são são todas as propriedades qe são herdadas.
Exemplo: a tag border, porque ela é uma tag que não é herdada (coisas estruturais).

OBS: O css funciona de cima pra baixo


--Módulo 4 (aula 2) Especificidade
-Quanto mais específico você for sobre o elemento que você quer aplicar aquela propriedade, ele que vai "ganhar" no CSS.
Ou seja, o mais específico sempre ganhar.

--Módulo 4 (aula 4) Unidades de medida absoluta

-medidas absolutas são medidas que não vão sofrer alterações por outros elementos do site
-Existe algumas unidades de medidas que a são consideradas absolutas (o pixel é uma delas).

-pt = unidade de medida absoluta, porém é utilizada mais nas fontes.
-in (polegadas) = também é considerada medida absoluta 
-cm e mm = é mais utilizado para imprimir 


--Módulo 4 (aula 5) Unidades de medida relativas (fonte) 1

-As unidades de medida que ela vão mudar de tamanho baseado nos elementos que tem na própria página.

-em = é baseado no font-size no item pai do elemento que estou botando botando.
Exemplo:
(HTML)

  <div class="box">
            <div class="box-interno ">
                Esta é uma frase 
            </div>
        </div>


(CSS) 
.box {
    font-size: 20px;

}
.box-interno {
    background-color: #999;
    width: 10em;
    height: 10em;
}

Percebe-se que o "box" é meio que o "pai" da box-interno, e no CSS a font-size do box é de 20px, então se eu for colocar a unidade de medida "em" na box-interno, "1em" vai ser igual a "20px", "2em" vai ser igual a "40px", "3em" vai ser igual a "60px", assim por diante...

-Rem = É como se fosse um "em" porém ele corresponde ao primeiro elemento da tela, ou seja, se eu coloco no "html" um font-size de 10px, e no h1 coloco "2rem" o meu h1 vai ter 20px.

-OBS: quando eu não especifico o font-size, por padrão (na maioria dos navegadores) vai ser 16px.


--Módulo 4 (aula 6) Unidades de medida relativas (fonte) 2

Quando coloca 50% no box, significa que ele vai pegar 50% do ESPAÇO DISPONIVEL.
OBS: uma div vai responde baseado no tamanho de cima dela.


ex (3-5 minutos)
ex: é um tipo de medida baseada no tipo de fonte que vc ta usando, ou seja, o tamanho dela vai ser o tamanho da altura da letra x na fonte que esta sendo aplicada naquele momento.
exemplo: ele vai pegar a letra x mesmo, vai medir a altura dela e vai definir essa altura como o tamanho da letra.

ch (6-final)
A ch é usada muito quando tem um box onde possui uma frase.
ch: todas as caracterias tem o mesmo espaço de largura, porém só funciona com fontes mono space.


--Módulo 4 (aula 7) Unidades de medida relativas (viewport)

vh (1 - 3:35 minutos)
Exemplo de vh: height: 100vh;
Oque significa: é 100% da altura disponivel para o usuário.
OBS: é muito usada para fazer um menu lateral por exemplo.


vw (3:35 - 5:55 minutos) 
"vw" vai ser 100% da LARGURA quue esta disponivel na tela.
Obs: 100 significa 100%, igualmente como 50 sigifica 50%, 1 significa 1%, etc...
IMPORTANTE: Pode variar muito conforme o tamanho da tela
IMPORTANTE: não é usado muito pra fonte, mas sim para posicionar estruturas gerais do site. 


vmin e vmax (6 - final)
vmin: ele vai identificar entre a altura e a largura quem é o menor pra usar como referência.
Obs: no caso do computador, na maioria das vezes o menor é a altura

vmax: o inverso do vmin 


--Módulo 4 (aula 8) O inherit  

inherit ele vai herder a força/propriedade do item a cima, ou seja, forçar uma herança.
Obs: é muito pouco utilizada.

Quero que o meu h1 dentro do box fique do mesmo tamanho original do meu box.

}


ou

--Módulo 4 (aula 9) Tipos de valores para cores
É possivel identificar uma cor de varias formas

hexadecimal (2 - 5:30 minutos)
Geralmente essas cores são representadas por um código de 6 caracteres hexadecimais (de 0 a F) precedido do caracter "#", por exemplo: '#68B46D'. 

Os 2 primeiros são correspondentes as tonalidades de vermelho
O 3 e 4 são correspondentes as tonalidade do verde 
O 5 e 6 são correspondentes as tonalidade do azul

0 Corresponde a NADA daquela tonalidade e F corresponde a TUDO daquela tonalidade


RGB (5:30 - 7 minutos)
Para definir o código RGB de uma cor vc coloca "rgb()" dentro do parênteses vc coloca números separados por vígura que podem ir de 0 ate 255.


HSL(7 - 7:50 minutos)
Mesmo procedimento do RGB, porém vai ser usado porcentagem.


Há também uma forma de vc definir uma cor baseado na cor de cima.
SITUAÇÃO: quero que o meu h1 fique com a borda da mesma cor que o item de cima.
RESOLUÇÃO: 

body {
    margin: 0;
    color: #f00;
}
H1 {
    border: 2px solid currentColor; 
}
     
EXPLICAÇÃO: em vez de colocar o nome da cor, pode simplesmente colocar currentColor, e isso gera um efeito de cascata.



--Módulo 5 (aula 2) Layout adaptativo vs layout responsivo

Layout não responsivel: é o Layout que não se adapta ao tamanho da tela.

Layout responsivel: é um Layout pensado para ir diminuindo de tamanho de acordo com o tamanho da própria tela.

Layout adaptativo: é um Layout que se adpeta ao tamanho da tela específico, porém ele não se adapta de forma fluida como o responsivel, ou seja, ele se renderiza de forma diferente dependendo do tamanho da tela.

OBS: na maioria dos sites vai ter que fazer um adeptativo, junto com um responsivel.



--Módulo 5 (aula 3) Viewport

O viewport é a área onde seu website aparece. É a área branca da janela quando você abre o browser. 
O viewport sempre vai ter o tamanho da janela. Mas a forma como os elementos são renderizados vai depender bastante do dispositivo.
